$dumpfile


Set file for writing simulation stimulus and results. Resulting file is a waveform dump in one of the formats supported by simulator. Results could be viewed in external viewer for example GTKWave.
argument decription
filename File for writing simulation results

Filename could be stored in a register and specified in simulation params.

Example:

initial begin
    $dumpfile("simulation.vcd");
    $dumpvars(0, top_tb);
end
