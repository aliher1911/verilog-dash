$dumpvars


Dumpvars enables writing of variables to a dump file. It can have two forms. If used wiithout arguments, it will write all design variables into a dump file. If used with arguments, it will write only specified variables and modules up to a required level down.
argument decription
level Number of levels of design to write. 0 - write all the variables in all modules down from specified module, 1 - write only variables defined in module etc.
module [, module] Module or variable name to write to dump. Multiple modules or variables could be specified.

Example:

initial begin
    $dumpfile();
    // will dump top level variables of test bench
    // as well as all variables inside uut module
    $dumpvars(1, top, top.uut);
end
