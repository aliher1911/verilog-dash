$strobe


Print formatted string to stdout. Outputs data every time it is executed at the end of current simulation cycle.
argument decription
format string Format string
a, b, ... Values to expand according to parameter string

Format string is identical to $display function.

Example:

initial begin
    en <= 0;      
    @(posedge clk);
    $strobe("Signal %d", en); // will produce 'Signal 1'
    en <= 1;
    @(posedge clk);
end
