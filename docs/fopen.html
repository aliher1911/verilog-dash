$fopen


Opens text file for writing. Up to 32 files could be open at the same time including standard output. Returned file handle has a single bit set corresponding to curreent file. Handles could be OR'ed together to make multifile handle and subsequentlyused to work with multiple files.
argument decription
filename File name

Return value

Returns integer multichannel descriptor for subsequent file operations or 0 if file can not be open.

Example:

integer logFile, logging;
intial begin
    logFile = $fopen("File1.log");
    // We should log to console and a file log
    logging = LogFile | 1;

    $fdisplay(logging, "Simulation starts");
    // ...
    $fclose(logFile);
end
