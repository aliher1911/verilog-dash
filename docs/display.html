$display


Print formatted string to stdout. Outputs data every time it is executed.
argument decription
format string Format string
a, b, ... Values to expand according to parameter string

Format string is interpolated to output string by replacing placeholders with remaining parameters. Each placeholder starts with % character which is followed by type.
Placeholder consts of %[width][.precision]format
format type
%b binary
%d decimal
%h hexadecimal
%o octal
%e real in the form of 3.1500e+00
%f real in the form of 3.15
%g real in f or e style depending on value and precision given
%v Signal with strength: Su - supply, St - strong, Pu - Pull, La - Large, We - Weak, Me - Medium, Sm - Small, Hi - Highz
%c character
%s string
%t time
%m hierarchical structure

Optional width defines minimum number of characters. Numbers are padded to the left with spaces or zeroes if width has a leading zero.
Optional precision defines how many digits to specify after decimal point for real numbers.
For more formatting details check c printf function (man 3 printf).

Example:

initial begin
    en <= 0;      
    @(posedge clk);
    $display("Signal %d", en); // will produce 'Signal 0'
    en <= 1;
    @(posedge clk);
end
