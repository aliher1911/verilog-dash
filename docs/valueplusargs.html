$value$plusargs


Reads a plusarg value into variable. Plusargs are added as + prefixed arguments passed to simulator. Plusargs are usually defined as +argument=value but = is not mandatory and +argumentvalue could be matched as well.
argument decription
format Format constiting of plusarg name followed by value format specifier. Name should contain = character if it is provided in plusarg.
variable variable to read value into

Task supports follwing format specifiers:
format type
%b binary
%d decimal
%h hexadecimal
%o octal
%e real in the exponential form (e.g. 3.1500e+00)
%f real in the decimal form (e.g. 3.15)
%g real in f or e style
%s string

Return value

Returns 1'b0 if no matching plusarg was found, 1'b1 if plusarg was found. If format can't be matched with string, task still succeeds.

Example:

      // run simulation with +rate=3.14 to pass the rate
      // or run without rate to use default value
real rate;
initial begin
      if ($value$plusargs("rate=%f", rate)) begin
         $display("Got rate %f", rate);
      end else begin
         $display("Can't find rate arg");
         rate = 1;
      end
end
