$fdisplay


Print formatted string to one or more files previously open with $fopen.
argument decription
mcd Multichannel descriptor to write message to
format Format string
a, b, ... Values to expand according to parameter string

Printing behaviour and format string is identical to $display function.

Example:

integer logfile;
      
initial begin
    logfile = $fopen('simulaition.log');
    en <= 0;      
    @(posedge clk);
    $fdisplay(logfile, "Signal %d", en); // will write 'Signal 0' to simulation.log
    en <= 1;
    @(posedge clk);
    $fclose(logfile);      
end
