$monitor


Print formatted string to stdout. Outputs data every time one of its arguments changes. Monitor could only be defined in initial or always blocks. There could be only one active monitor at the same time, setting new one will disable the other.
argument decription
format string Format string
a, b, ... Values to expand according to parameter string

Format string is identical to $display function.

Example:

initial begin
    $monitor("Iteration %i at %t", i, $time);
end
