$test$plusargs


Tests if plusarg is present. Plusargs are added as + prefixed arguments passed to simulator. Plusargs are usually defined as +argument=value but = is not mandatory.
argument decription
prefix Plusarg name prefix. Any arg starting from prefix string will match, so to avoid longer names matching = sign could be included in the prefix to match.

Return value

Returns 1'b0 if no matching plusarg was found, 1'b1 if plusarg was found.

Example:

      // run simulation with +rate=3.14
initial begin
      if ($test$plusargs("rate")) begin
          $display("Found rate"); // will print Found rate
      end
      if ($test$plusargs("ra")) begin
          $display("Found rate"); // will print Found rate
      end
      if ($test$plusargs("rate=")) begin
          $display("Found rate"); // will print Found rate
      end
      if ($test$plusargs("rate=4")) begin
          $display("Found rate"); // will not be executed
      end
end
